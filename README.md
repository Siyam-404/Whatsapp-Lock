# Installation
## Linux
```
sudo apt update 
sudo apt upgrade
sudo apt install git nodejs npm python-pip
git clone https://github.com/STARK-404/Whatsapp-Lock
cd Wp-Lock
pip install -r requirements.txt
python Run.py
```
## Termux
```
pkg update 
pkg upgrade 
pkg install git nodejs python
git clone https://github.com/STARK-404/Whatsapp-Lock
cd Wp-Lock
pip install -r requirements.txt
python Run.py
```
